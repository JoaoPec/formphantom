Salve seu perfil

Abra a vaga

O FormPhantom detecta os campos automaticamente.

3

Revise, ajuste o que quiser e deixe o fantasma trabalhar.

Instalar no Chrome (modo desenvolvedor)

$ git clone https://github.com/JoaoPec/formphantom.git $ cd formphantom
  1. Abra o Chrome e vá para chrome://extensions/
  2. Ative o Modo do desenvolvedor.
  3. Clique em Carregar sem compactação.
  4. Selecione a pasta formphantom.
  5. O ícone do FormPhantom aparecerá na barra de ferramentas.
