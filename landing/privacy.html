FormPhantom

Política de Privacidade

Última atualização: 25 de junho de 2026

O FormPhantom é uma extensão de navegador projetada para ajudar candidatos a preencher formulários de vagas de forma mais rápida e organizada. Levamos a privacidade dos seus dados a sério.

1. Dados que coletamos

A extensão armazena localmente no seu navegador as informações que você mesmo insere no perfil, como:

  • Dados pessoais (nome, e-mail, telefone, endereço);
  • Informações profissionais (experiência, formação, habilidades);
  • Texto extraído de currículos em PDF que você anexar voluntariamente.

Esses dados são salvos apenas no armazenamento local do Chrome (chrome.storage.local) e não são enviados para nossos servidores.

2. Como usamos os dados

Os dados são usados exclusivamente para preencher formulários de candidatura nas páginas que você visita. Nenhuma informação é vendida, compartilhada ou utilizada para publicidade.

3. Permissões do navegador

O FormPhantom solicita as seguintes permissões para funcionar corretamente:

  • storage: para salvar seu perfil localmente no navegador;
  • activeTab: para atuar apenas na aba que você está usando no momento da ação;
  • scripting: para detectar e preencher os campos dos formulários;
  • tabs: para identificar a página ativa e executar o preenchimento;
  • host_permissions (<all_urls>): para que a extensão funcione em qualquer site de vagas, sem limitação de domínio.

4. Currículos em PDF

Quando você anexa um PDF, a extensão extrai o texto localmente usando a biblioteca PDF.js. O arquivo e o texto extraído permanecem no seu navegador e não são enviados para lugar nenhum.

5. Segurança

Como os dados ficam apenas no seu dispositivo, você tem controle total sobre eles. Você pode excluir seu perfil a qualquer momento diretamente na extensão.

6. Alterações nesta política

Podemos atualizar esta política ocasionalmente. Qualquer mudança será publicada nesta página com a data da última atualização.

7. Contato

Em caso de dúvidas sobre privacidade ou sobre o funcionamento da extensão, abra uma issue no repositório do GitHub.
